# Atenea

Atenea es una plataforma web para la gestión de bibliotecas. Permite administrar libros, autores y el control de préstamos de manera sencilla.

## Características
- Registro y administración de libros.
- Gestión de autores.
- Control de préstamos y devoluciones.
- Interfaz web fácil de usar.

## Estructura del proyecto
- **/api** – Lógica del backend.
- **/controllers** – Controladores de la aplicación.
- **/models** – Modelos de datos.
- **/public** – Vistas y archivos accesibles públicamente.
- **/db** – Archivos de base de datos.
- **/assets** – Recursos como imágenes y estilos.

## Cómo usar
1. Clona el repositorio:
2. Coloca el proyecto en tu servidor local (XAMPP, WAMP, Laragon, etc.).
3. Importa la base de datos desde la carpeta db.
4. Configura la conexión a la base de datos si es necesario.
5. Abre el proyecto en tu navegador

## Requisitos
Servidor con soporte PHP.
Servidor MySQL/MariaDB para la base de datos.
